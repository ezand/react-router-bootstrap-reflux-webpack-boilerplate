# React and friends boilerplate

Used technologies:  

- React
- ES6
- Webpack
- Babel
- Bootstrap SASS
- React Bootstrap
- React Router
- React Router Bootstrap
- Reflux

## How to build

```
npm install
```

Open browser with http://localhost:8080/ or http://localhost:8080/webpack-dev-server/

## How to start dev server

`npm start`
